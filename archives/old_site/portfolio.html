Portfolio

  • Composing Compounds

    A game experience built to bring the periodic table out of a realm of abstract knowledge to relatable items in the world. The target age is a middle school audience.

    After building a compound it will tell you the compound that you built. While subtly introducing concepts like lewis dot structures as a mechanic, and groups of elements through color scheme. On this project I was the Creative Director, Designer, and Programmer.

  • Little Fields

    A platforming game for children that focuses on teaching avoidance of conflict. On this project I served as a Programmer and helped write the first iteration of the character controller, and wrote the AI code for 3 different enemies. Tools used: Unity, SVN, Trello.

  • Zarathustra

    A randomly generated roguelike with a Lovecraftian theme. On this semester project I served as the Producer to a team of 12 students. Tools used: Unity, SVN, Trello.

  • Raaargh

    Raaargh is a 3D monster control game built in UDK and deployed on iOS. The game was built as a demo for class not for actual market deployment. For the project I was a Programmer and built the back end systems along with being responsible for iOS deployment. Tools used: UDK, iOS, Visual Studio, SVN.

  • Choker

    Board game that combines chess and poker elements to create an abstract strategy game of perfect information. I designed the board myself, and the gameplay was a collaborative effort. Tools used: Photoshop.
